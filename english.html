    名詞

    人や物の名称を表す

    可算名詞：cat, dog, bird, cream, desk, pen, pencil, book, apple, banana, computer, camera など

    不可算名詞：bread, water, paper, iron, suger, snow, rice, wood など

    I, you, he, she, it, we, they, my, your, his, her, its, our, their, me, him, us, them, mine, yours, hers, ours, yours, theirs, everyone など

    動詞

    動作や状態を表す

    be動詞：am, are, is, was, were

    一般動詞：come, eat, go, have, help, like, live, look, meet, open, play, read, run, see, speak, study, swim, walk, watch など

    助動詞

    動詞の直前に付いて、その動詞に色々な意味を持たせる

    can, may, will, would, do(疑問/否定) など

    冠詞

    名詞の直前に付いて、どのような名詞か説明する

    a, an, the

    a：後の音が子音（母音以外）

    an：後の音が母音（a/i/u/e/o）

    the：後の単語が子音から始まるならザ、母音ならジと発音する。

    形容詞

    名詞・代名詞を修飾する

    good, happy, new, young, little, beauiful, pretty など

    副詞

    動詞・形容詞・副詞・文全体を修飾する

    動詞の直前・be動詞や助動詞の直後に書く。

    yes, no, not, too, very, how, usually, sometimes, together, well, today, there など

    前置詞

    名詞・代名詞の直前に置かれ、時・場所・方向・手段などを表す

    in, at, on, with, inside, before, after, by, from, beside, of, for など

    in (場所)〜の中に[で,の]、(時間)〜の間に
    at (場所/時間])〜に[で]
    on 〜の上に[で,の]、(日付/曜日)〜に
    with 〜と(一緒に)、〜で(〜を使って)
    inside (場所)〜の内部(中)で[へ,で]
    before 〜よりも前(先)に
    after 〜のあとで[に]、〜してから、(ある時間)経ってから
    by 〜によって、(通信/移動手段)〜で
    from 〜から(の)
    beside 〜のそばに、〜の近くに
    of 〜の
    for 〜のために

    接続詞

    語・句・節などを結ぶ

    and, or, but, that, if など

    間投詞

    驚き、怒り、喜びなどの感情を表す

    oh, hi, wow など

    複数形

    複数形

    複数形の作り方

    1. ほとんどの名詞は語尾に s を付ける

    book → books

    pen → pens

    2. 語尾が s, x, sh, ch の単語は es を付ける

    bus → buses

    box → boxes

    dish → dishes

    watch → watches

    3. 語尾が 子音字 + y の単語は y を i に変えて es を付ける

    city → cities

    ※ 子音字：a, i, u, e, o 以外のアルファベット

    4. 語尾が f または fe で終わる単語は f または fe を v に変えて es を付ける

    leaf → leaves

    knife → knives

    5. 語尾が 子音字 + o の単語には es を付ける

    potato → potatoes

    tomato → tomatoes

    hero → heroes

    ※例外：pianos, photos, solos など（他の語を短縮している）

    6. 不規則変化をする名詞

    man → men

    woman → women

    child → children

    foot → feet

    7. 変化をしない名詞

    1. 群れで生活する生物

    sheep, deer, fish など

    2. ese で終わる国民

    Japanese, Chinese など

    複数形の発音

    1. ほとんどの単語は [z]

    boys, bags, pencils, schools など

    2. 語尾が [p] [t] [f] [k] の単語は [s]

    roofs, cups, cakes, bats, pots など

    覚え方：プ～と吹く（プトゥフク）

    ※ cats, bats, hats のような単語は、発音は [ツ] になる。

    3. 語尾が [s] [z] [ʃ] [ʧ] [ʤ] の単語は [iz]

    boxes, classes, dishes, benches, bridges など

    覚え方：鈴7時（スズシチジ）

    3人称単数現在形の発音

    1. ほとんどの単語は [z]

    plays, goes, comes, enjoys, studies など

    2. 語尾が [p] [t] [f] [k] の単語は [s]

    helps, hits, likes, makes など

    ※ hits のような単語は、発音は [ツ] になる。

    現在進行形

    現在進行形の作り方

    1. ほとんどの動詞は語尾に ing を付ける

    play → playing

    speak → speaking

    2. 語尾が 子音字 + e の単語は e をとって ing を付ける

    write → writing

    take → taking

    ※ 語尾が 母音字 + e の単語は e をとらずに ing をつける（see → seeing など）

    3. 語尾が 短母音 + 子音 の動詞は最後の子音字を重ねて ing を付ける

    run → running

    begin → beginning

    swim → swimming

    ※ 短母音：連続していない母音

    4. 語尾が ie で終わる動詞は ie を y に変えて ing を付ける

    lie → lying

    tie → tying

    die → dying

    ※ dye は die と区別するため、dyeing とする。

    過去形/過去分詞

    過去形/過去分詞の作り方

    1. ほとんどの動詞は語尾に ed を付ける

    play → played

    walk → walked

    2. 語尾が e で終わる動詞は d を付ける

    like → liked

    use → used

    3. 語尾が 子音字 + y で終わる単語は y を i に変えて ed を付ける

    study → studied

    4. 語尾が 短母音 + 子音字 で終わる動詞は最後の子音字を重ねて ed を付ける

    stop → stopped

    ※ 短母音：連続していない母音

    過去形/過去分詞の発音

    1. ほとんどの単語は [d]

    played, enjoyed, arrived, closed, called など

    2. 原型の語尾が [t] [d] で終わる単語は [id]

    wanted, started, visited, ended, skated, collected など

    3. 語尾が [s] [p] [f] [k] [ʃ] [ʧ] で終わる単語は [t]

    liked, looked, picked など

    覚え方：スープ吹くシチュー

    単語の違い

    some / any

    単語 意味 使う時
    some いくつかの 肯定文
    any 疑問文・否定文

    someもanyも複数を表す語のため、名詞が複数形になる。

    many / much / a lot of

    単語 意味 使う時
    many たくさんの 数で数えるもの（複数形があるもの）
    much 量で数えるもの（複数形がないもの）
    a lot of 両方使用できる

    see / look / watch

    単語 意味 使う時
    see 見る 自然に目に入る
    look 目を向ける
    watch 動いているものを見る

    映画館で見る映画は大画面のため自然に目に入るということで、See a movie を使う（家ではWatch）。

    talk / tell / speak / say

    単語 意味 使う時
    talk 話す 人と話し合う(会話)
    tell 情報の伝達(教える)
    speak 人に向かって話す
    say 言う 口から単語を出す

    very / so

    単語 意味 使う時
    very とても 感情的ではない
    so 感情的
